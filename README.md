Roblox Auto Reconnect Tools

This tool can detect disconnected, crashing, even freezing. 
Roblox Auto Reconnect Tools also OPEN SOURCE so you can modified it

How To Use
1. Download Termux from Fdroid : (https://f-droid.org/en/packages/com.termux/)
2. Open termux and then [pkg update -y && pkg upgrade -y && pkg install -y curl wget bash coreutils procps termux-tools] if there asking Y/N/D type Y for all.
3. termux-setup-storage
4. su 
5. exit
6. curl -o ~/roblox_reconnect.sh https://raw.githubusercontent.com/wardz25/reconnect-rblx/refs/heads/main/roblox_reconnect.sh
7. chmod +x ~/roblox_reconnect.sh
8. bash ~/roblox_reconnect.sh
