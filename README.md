# ⚒️ Roblox Auto Reconnect Tools

This tool can detect disconnected, crashing, even freezing.

Roblox Auto Reconnect Tools is OPEN SOURCE

# ⚙️ How To Use
It might be complex but isn't hard trust me :D

## 📥 1. Download Termux from Fdroid 
Termux : [Click Here](https://f-droid.org/en/packages/com.termux/)

## 📂 2. Open termux and then copy and paste below!
```
pkg update -y && pkg upgrade -y && pkg install -y curl wget bash coreutils procps termux-tools
```
Note : Just type Y if there asking for Y/N/D
## 🗂️ 3. Enable Permission for Storage
```
termux-setup-storage
```
## 📱 4. Enable root in your device and then check on termux with this command
```
su
```
## 5. Almost there!
```
curl -o ~/roblox_reconnect.sh https://raw.githubusercontent.com/wardz25/reconnect-rblx/refs/heads/main/roblox_reconnect.sh
## 6. Enable Permission for the script
```
chmod +x ~/roblox_reconnect.sh
```
## 7. Run the Script!
```
bash ~/roblox_reconnect.sh
````
